## 2025-05-14 - Label and Focus-Visible Improvements
**Learning:** Explicit association between <label> and <input> using 'for'/'id' attributes is critical for screen readers and significantly improves the hit area for all users. Using :focus-visible instead of :focus ensures that keyboard users have a clear indicator without affecting mouse users' visual experience.
**Action:** Always verify label associations and implement a consistent :focus-visible strategy in every project to ensure high accessibility standards.
