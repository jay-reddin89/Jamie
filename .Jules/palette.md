## 2025-05-14 - Semantic Form Association
**Learning:** Explicitly associating `<label>` elements with their corresponding inputs using `for` and `id` attributes is a fundamental but often overlooked accessibility win. It not only assists screen reader users by providing clear context for form fields but also improves the hit target area for all users (clicking the label focuses the input).
**Action:** Always verify that every form input has a uniquely associated label, especially in custom-styled dashboards where semantic structure might be sacrificed for aesthetics.
