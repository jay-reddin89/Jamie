## 2025-05-15 - Improving Accessibility and Keyboard Navigation
**Learning:** Associating labels with inputs using the `for` attribute and providing clear `:focus-visible` indicators significantly improves the experience for keyboard and screen reader users, making the form more intuitive and accessible. Adding global keyboard listeners (like 'Escape' for modals) aligns with standard UX patterns and user expectations.
**Action:** Always ensure form elements are properly labeled and that all interactive components have distinct focus states and relevant keyboard shortcuts.
