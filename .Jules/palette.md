## 2025-05-15 - Enhancing Form Accessibility and Keyboard Interactivity
**Learning:** Standardizing form behavior (wrapping in <form>, using label for, and :focus-visible styles) significantly improves accessibility and allows for intuitive 'Enter' key submission without complex JS logic.
**Action:** Always wrap input groups in a semantic <form> and ensure label-input associations are explicit for both a11y and hit-area improvements.
