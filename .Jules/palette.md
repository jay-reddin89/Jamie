## 2025-05-14 - Enhanced Form Accessibility and Clarity

**Learning:** Explicit association between `<label>` and `<input>` using `for` and `id` attributes is critical for screen readers and significantly improves the hit target size for all users, particularly on touch devices. Marking mandatory fields with both visual (`*`) and programmatic (`aria-required`) indicators provides essential guidance and prevents user frustration.

**Action:** Always ensure `<label>` elements are linked to their inputs and that required fields are clearly identified using both visual and ARIA cues.
