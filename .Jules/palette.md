## 2025-05-15 - [Form Accessibility and Interactivity]
**Learning:** Wrapping input groups in a semantic `<form>` element and correctly associating `<label>` tags with `for` attributes drastically improves both accessibility (screen readers, click targets) and usability (Enter-to-submit).
**Action:** Always wrap input groups in a `<form>` and ensure every input has an associated `<label>` with a `for` attribute matching the input's `id`.
