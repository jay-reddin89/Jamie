## 2025-03-24 - Onboarding Form Accessibility
**Learning:** Explicit association between <label> and <input> using 'for' and 'id' attributes is critical for screen readers and significantly improves the hit target size for all users, particularly on touch devices. Additionally, using native 'required' attributes provides immediate browser-level validation that complements custom JS logic.
**Action:** Always ensure every form input has a uniquely identifiable ID and a corresponding label with a matching 'for' attribute. Include 'required' markers (*) and attributes for all mandatory fields.
