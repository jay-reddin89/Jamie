## 2025-05-15 - Form Wrapping and Keyboard Accessibility
**Learning:** Wrapping input collections in a `<form>` element is a high-impact, low-cost UX win for accessibility and efficiency. It enables native "Enter to Submit" behavior, which is a core expectation for keyboard users. Combined with proper `<label>` associations via the `for` attribute and `:focus-visible` styles, it significantly improves the interface's predictability and operability.
**Action:** Always wrap onboarding and settings inputs in a `<form>` element and ensure labels are correctly linked to their inputs.
