## 2025-05-14 - Accessible Form Feedback & Focus
**Learning:** Combining semantic labels (via 'for' attribute), visual 'required' indicators, and high-contrast error notifications significantly improves form usability and accessibility. Additionally, using ':focus-visible' ensures keyboard users have clear visual feedback without affecting mouse users' experience.
**Action:** Always associate labels with inputs using 'for'/'id' and implement distinctive error states for the notification system. Use ':focus-visible' for interactive element focus states.
