## 2025-05-15 - Improving Accessibility of Input Forms

**Learning:** Explicit association between `<label>` and `<input>` using `for`/`id` attributes is critical for screen readers and significantly improves the hit area for all users, particularly on mobile devices.

**Action:** Always ensure every form input has a corresponding label with a matching `for` attribute and that decorative icons are hidden from screen readers using `aria-hidden="true"`.
