## 2025-05-14 - Label-Input Association and Focus States
**Learning:** In dashboard interfaces with dense forms, explicit label-input association via the 'for' attribute significantly improves mobile UX by expanding tap targets. Additionally, :focus-visible provides a non-intrusive way to ensure keyboard accessibility without cluttering the UI for mouse users.
**Action:** Always ensure <label> elements have 'for' attributes matching input 'id's, and implement :focus-visible styles for all interactive elements.
