## 2025-05-15 - Enhancing Form Interactivity and Accessibility
**Learning:** Wrapping input fields in a `<form>` element combined with `type="submit"` buttons provides a significant keyboard accessibility win ("Enter to Submit") with minimal code. Using `:focus-visible` ensures that these interactive elements are clearly identifiable during keyboard navigation without adding clutter for mouse users.
**Action:** Always prioritize semantic HTML forms for data entry sections and ensure all interactive elements have clear focus indicators and associated labels.
